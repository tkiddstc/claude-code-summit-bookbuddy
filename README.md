# BookBuddy

A simple book review app for the **Claude Code Summit** workshops.

## Setup

```bash
git clone <repo-url>
cd bookbuddy
npm install
npm run seed
npm start
```

Open [http://localhost:3000](http://localhost:3000).

## What's in here

Express.js app with EJS templates and SQLite (via better-sqlite3). Six sample books with reviews pre-loaded by the seed script.

- Browse books with average ratings
- View book details and reviews
- Add, edit, and delete books
- Submit and remove reviews
